# ARTIKLE STORE

## Descripción
ARTIKLE STORE es una página web diseñada para ofrecer asesoría y productos tecnológicos. La página incluye funcionalidades de inicio de sesión, registro, visualización y adición de reseñas, y un sistema de navegación por pestañas.

## Estructura del Proyecto
El proyecto está compuesto por los siguientes archivos:
- `ARTIKLE.html`: Contiene la estructura HTML de la página.
- `ARTIKLE.css`: Contiene los estilos CSS para la página.
- `ARTIKLE.js`: Contiene la lógica JavaScript para la interacción de la página.

## Funcionalidades
### Inicio de Sesión y Registro
- **Inicio de Sesión**: Permite a los usuarios iniciar sesión en la página.
- **Registro**: Permite a los nuevos usuarios registrarse en la página.

### Reseñas
- **Visualización de Reseñas**: Muestra las reseñas de los usuarios.
- **Añadir Reseña**: Permite a los usuarios añadir nuevas reseñas.

### Navegación
- **Pestañas**: Sistema de navegación por pestañas para acceder a diferentes secciones de la página.
